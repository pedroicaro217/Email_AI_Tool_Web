# Email Marketer Web (Arquiteto de E-mails)

Aplicação web em Python (Flask) e IA (Google Gemini) para automação de campanhas de e-mail marketing.

Este projeto, parte do meu portfólio, demonstra uma arquitetura de serviço web completa, incluindo:
- Backend em Flask
- Banco de Dados SQLite
- Geração de conteúdo HTML via API de IA
- Envio de e-mails em massa via SMTP
